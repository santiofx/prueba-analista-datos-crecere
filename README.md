# Análisis de negociación de deudas en mora prolongada (>180 días)

Este repositorio contiene el desarrollo completo de la prueba técnica para el rol de Analista de Datos.

## Contenido del proyecto

- Notebook con el análisis exploratorio y cálculo de métricas.
- Dataset utilizado en el ejercicio.
- Tablas de resultados exportadas para visualización.
- Resultados utilizados en el dashboard.

## Estructura del repositorio

- data/: base de datos original
- notebooks/: análisis en Python (Jupyter Notebook)
- outputs/: tablas resumen generadas
